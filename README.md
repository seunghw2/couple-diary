# 투데이 (Two-day) — 커플 교환일기 앱

> two + today. 우리 둘의 오늘을 함께 기록하는 교환일기.

커플이 함께 쓰는 **교환일기 앱**입니다. 둘 다 써야 상대의 글이 열리고, 사진과 데이트 장소를 남기고, 시간이 지나면 "n년 전 오늘"로 되돌아보는 추억 타임캡슐.

## 핵심 컨셉
- 📝 **상호 공개** — 둘 다 오늘 일기를 써야 상대 글이 열림
- 🖼️ 사진 첨부 · 📍 데이트 위치 저장
- ⏱️ 수정은 작성 몇 시간 뒤부터
- 🧩 템플릿(빈칸 채우기)으로 쉽게 작성
- ⭐ 데이트 점수 + 😊 오늘의 기분
- 📅 달력에서 지난 일기 열람
- ⏳ "n년 전 오늘" · 기념일 회고 (추억 타임캡슐)
- 🎀 아기자기한 일기장 감성 (게임 요소 없음)
