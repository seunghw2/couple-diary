# 02 · 일기 댓글

## 목적
상대(또는 본인)의 그날 일기에 **댓글로 대화**한다. 교환일기의 상호작용을 이모지 반응보다 한 단계 깊게.

## 커플 규칙 정합성 (중요)
- **상호 공개 규칙 존중**: 그날 일기는 둘 다 작성해 "열린" 뒤에만 댓글 가능. 잠금(한 명만 작성) 상태에선 댓글 숨김.
- 일기 본문 **수정 지연(몇 시간)** 규칙과 **댓글은 별개** → 댓글은 즉시 작성/표시.
- 이모지 하트 반응과 **병행**(반응=가벼움, 댓글=대화).

## 동작
1. 일기 상세 하단에 **댓글 목록 + 입력창**
2. 작성 시 작성자 아바타·이름·색으로 구분(승환/수정)
3. 시간순 정렬, 실시간(또는 폴링) 갱신
4. 새 댓글 시 상대에게 알림(로컬/인앱 브리지)

## 데이터
- `comment { id, entryId, authorId, text, createdAt, updatedAt }`
- 조회: entryId 기준 목록
- 권한: 커플 둘만 읽기/쓰기

## 화면
- 일기 상세 하단 댓글 섹션(목록 + 입력 바)
- 빈 상태: "첫 댓글을 남겨보세요" 안내

## 엣지케이스
- 빈/공백 댓글 방지
- 본인 댓글만 수정/삭제
- 길이 제한(예: 500자)
- 잠긴 일기에서는 댓글 진입 불가
- 삭제된 일기의 댓글 처리(함께 정리)
