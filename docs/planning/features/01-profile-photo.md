# 01 · 프로필 사진 변경

## 목적
각자 자신의 프로필 사진을 설정/변경한다. 홈 헤더에 **두 사람 프로필이 나란히** 노출되어 "우리 둘" 정체성을 강화한다. (커플감 강화 레버)

## 동작
1. **설정 → 내 프로필 → 사진 변경**
2. 소스 선택: 카메라 촬영 / 갤러리에서 선택
3. 정사각형 크롭(원형 마스크 미리보기)
4. 저장 → 내 아바타 즉시 갱신, **상대 화면에도 반영**(공유 커플 공간)
5. 홈 헤더·일기 상세 말풍선·댓글 등 아바타가 쓰이는 모든 곳에 반영

## 데이터
- `user.avatarUrl` (없으면 기본 아바타)
- 커플은 서로의 프로필을 조회 가능 → 홈 헤더에 둘 다 표시
- 이미지 업로드: 리사이즈(예: 512px) 후 저장, 용량 제한(예: 5MB)

## 화면
- 설정 · 프로필 편집 (사진 + 이름/색 지정 자리)
- 홈 헤더: 두 아바타 나란히 + 커플 D-day

## 엣지케이스
- 기본 아바타: 이니셜 또는 이모지 플레이스홀더
- 업로드 실패/네트워크 오류 → 롤백 + 안내
- 큰 용량/비지원 포맷 → 거절 안내
- 크롭 취소 시 원상 복구

## 커플 규칙 정합성
- 프로필은 상호 공개(잠금 규칙과 무관, 항상 표시)
- 이후 "각자 색(승환색/수정색)" 지정과 연동 가능 → 아바타 테두리·말풍선 색
