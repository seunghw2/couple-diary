# 36. 일기·편지 수정은 24시간 이내에만

## 요청
일기 수정과 편지(답장) 수정을 작성 후 24시간 이내에만 가능하도록.

## 반영
### 일기
- 기존에도 수정 제한 창이 있었으나 **3시간**이었음 → **24시간**으로 변경(`EDIT_WINDOW_HOURS 3 → 24`).
- 프론트는 이미 `editable` 플래그로 수정 버튼/화면을 막고 있었고, 안내 문구의 "3시간"을 "24시간"으로 수정.

### 편지(답장)
- 기존엔 **시간 제한이 아예 없어** 봉인 후 언제든 수정 가능했음.
- 이제 **봉인(제출) 후 24시간 이내에만 수정** 가능. 초과 시 서버가 막음(403).
- today 응답에 `myAnswerEditable` 플래그 추가 → 24시간 지나면 답장 카드의 '수정' 버튼이 사라짐(대기·열람 상태 공통).

## 동작 정리
- 일기: 작성 후 24시간 안에만 고칠 수 있고, 지나면 수정 화면에서 막힘.
- 편지: 답장 봉인 후 24시간 안에만 고칠 수 있고, 지나면 '수정' 버튼이 사라지고 서버도 거부.

## QA (백엔드 E2E)
- 백엔드 컴파일 0, 부팅 0 에러 / 프론트 tsc 0.
- 답장 봉인 → 24h 내 수정 성공, `myAnswerEditable=true` 확인.
- 봉인 시각을 25시간 전으로 조작 → `myAnswerEditable=false`, 수정 시도 시 **403(Q002 "답장은 24시간 이내에만 수정할 수 있어요")**, DB 원문 변경 없음 확인.
