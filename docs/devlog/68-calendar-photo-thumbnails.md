# 68 · 달력에 저장 사진 표시 · 개수 배지/스티커 문구 제거

## 변경
1. **달력 셀** — 하트 스티커 대신, 그날 저장한 **사진이 있으면 그 이미지를 원형**으로 보여줌. 사진이 없는 날은 기존 코럴 하트 스티커 유지.
2. **사진 개수 배지 제거** — 셀 우상단에 뜨던 숫자(6/4/3…) 삭제.
3. **일기 상세** — "이 날의 스티커를 획득했어요!" 하트 원 섹션 제거.

## 구현
- **백엔드**: `MonthEntrySummary`에 `thumbUrl` 추가. `month()`가 그날 첫 실제 사진(url 있는)을 서명해 채움.
- **프론트**: `CalendarGrid`가 `e.thumbUrl` 있으면 `PhotoThumb`(원형)로 렌더, 없으면 하트 스티커 폴백. 개수 배지 블록 삭제. `entry/[date].tsx`에서 획득 스티커 섹션 삭제.
