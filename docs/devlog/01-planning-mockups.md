# 01 · 기획 & 목업 스토리보드

**날짜**: 2026-07-04
**목표**: 커플 교환일기 앱의 방향(컨셉·색·구조·전체 화면)을 목업으로 확정.

## 한 일
- 앱 컨셉 정의: 상호 공개(둘 다 써야 열림)·사진·데이트 위치·별점·기분·템플릿·수정 지연·달력 열람·"n년 전 오늘" 회고. 이름 **투데이(Two-day)**, 커플 승환·수정.
- 방향 탐색을 서브에이전트로 HTML 목업 제작하며 반복:
  - 홈 방향 4안(스티커달력/종이다이어리/폴라로이드피드/타임라인) → **사진 캘린더(썸원 참고)** 채택
  - 색 방향 여러 라운드 → **워밍 코럴 & 크림** 확정(듀오톤 블루·세이지·라벤더 등은 반려)
  - 자유(질문 픽) 작성 모드 설계: 먼저 쓴 사람이 질문 8개 중 3개 선택 → 상대는 그 범위에서 답
- 카카오 로그인~설정까지 **A to Z 전체 화면 스토리보드**(22화면) 작성.
