# 50. 내 사주 깊이 보강 (990식 근거·캐릭터, AI 없이)

## 요청
- 다른 사주앱(990 사주아이) 스크린샷 공유 — "이런 느낌으로 가능할까?"
- 990 특징: 강렬한 은유 캐릭터화 + 원국 데이터 실제 근거 노출("금 0개 전무" 등) + 개운법 + 팩폭 톤(1인 유료 개인 사주).
- 결정: **AI 없이(오프라인·결정론 유지) 고정 템플릿 보강** + **부드러운 개인화 톤**(팩폭·살 이름 나열 배제). 대상은 개인 사주(내 사주).

## 반영
### 백엔드 (라이브 배포)
- **반전 매력 한 줄(twist)**: 일간마다 겉/속 반전 캐릭터 카피. 예 "겉은 잔잔한 호수 같아도, 속엔 큰 꿈이 넘실대는 사람이에요."
- **"오행으로 본 나"(ohaengInsight)**: 내 원국의 **가장 강한 기운**과 **부족한(0개) 기운**을 짚어 근거를 보여주고, 부족한 기운에 대한 부드러운 보완법(색·산책·소품 등)을 성장 프레임으로 제시. 990의 "근거+개운법"을 겁주지 않는 톤으로.
  - 예(임수): "내 사주엔 목 기운이 가장 넉넉하게 흘러요… 반면 화·금 기운은 아직 여백으로 남아 있어요… 따뜻한 붉은 톤이나 햇살 아래 산책 같은 화의 기운을 곁에 두면 균형이 살아날 거예요."
  - 원국 오행 분포 기반이라 사람마다 달라짐(고정 캐릭터 + 개인 원국 = 개인화).
- DTO: `PersonalResult`에 `twist`, `ohaengInsight` 추가.

### 프론트
- 일간 카드: 한 줄 소개 아래 **반전 한 줄** 강조.
- "오행 분포" → **"오행으로 본 나"**: 막대 아래 강·약 기운 해설 박스.

## 톤 원칙
- 부정·직설·팩폭 배제, 살(煞) 이름 나열 안 함. "부족" 대신 "여백", 보완은 "채워가면 좋아요" 프레임.
- 전 콘텐츠 결정론(같은 원국=항상 같은 결과), 오프라인, API 비용 0.

## 검증
- 백엔드 clean compile / 프론트 tsc 0.
- 운영 백엔드에 유효 JWT(임수 유저)로 `/api/saju/me` 호출 → twist·ohaengInsight 정상.

## 후속 여지
- 더 깊게(십성·신살 계산 후 다층 소제목)나 AI 생성으로 990 수준까지는 별도 결정 필요(비용·톤 통제 동반).
