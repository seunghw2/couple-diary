# 64 · 십성 기반 '타고난 기질' (개인 사주 심화)

개인 사주가 '일간이 같으면 성격 설명이 똑같이' 나오던 약점을, **십성(十星)** 을 넣어
사람마다 달라지게 했다. (서브에이전트로 명리 규칙 검증 후 구현)

## 동작
개인 사주 화면 '오행으로 본 나' 아래에 **타고난 기질** 카드가 생겼다.
- 일간을 기준으로 나머지 일곱 글자의 기운을 다섯 묶음(비겁·식상·재성·관성·인성)으로
  세어, **가장 우세한 기질** 하나를 뽑아 보여준다.
- 유형명 + 키워드 칩 + 한 줄 설명. (예: 임수 근육남친 → 🎨 표현이 풍부한 기질 / 표현력·창의력…)
- 같은 일간이어도 오행 분포가 다르면 다른 기질이 나온다(요정체리 을목 → 🚩 주관이 뚜렷한 기질).

## 근거
- 십성 판정은 오행 상생·상극 관계로 결정(비겁=같은 오행, 식상=일간이 생, 재성=일간이 극,
  관성=일간을 극, 인성=일간을 생). 규칙은 갑목·계수 일간으로 교차검증해 표준과 일치 확인.
- 간이 버전이라 지장간·용신은 생략, 동점은 세력이 큰 월지 소속 묶음을 우선. '성향 참고' 수준.
