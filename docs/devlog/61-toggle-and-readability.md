# 61 · 알림 토글 클래식 + 사주 설명 가독성(B)

목업에서 고른 두 가지를 반영했다. (목업: `docs/design/notif-toggle-styles.png`,

## 알림 토글 → 클래식(1번)
- 기본 RN Switch가 플랫폼마다 색이 제각각(웹에서 초록 노브)이라 커스텀 `Toggle` 컴포넌트로 통일.
- 코럴 트랙 + 흰 노브의 iOS풍. 알림 설정·개발자도구 '테스트 계정 제외'에 함께 적용.

![알림 토글](captures/61-toggle-classic.png)

*알림 설정 — 코럴 클래식 토글*

## 사주 설명 가독성 → B(오행 근거 박스 분리)
- 우리 궁합 항목 설명이 한 덩어리로 빽빽했던 걸, **오행(사주) 근거만 '사주로 보면' 연한 코럴
  박스**로 떼어내 강조. 일반 설명은 문단으로.
- 근거가 있는 항목(첫끌림=일간 상생/상극, 성장=오행 상보)에만 박스가 붙는다.
- 본문 글자·줄간격도 살짝 키워 읽기 편하게.

![사주 가독성](captures/61-saju-readability.png)

*우리 궁합 — '사주로 보면' 근거 박스 분리*
