# 65 · 달력 월 이동 로딩 최소화

다른 달로 넘길 때 카드가 스피너로 통째 비던 걸 없앴다. (선택: A 인접 월 프리페치 + B 그리드 골격 항상 표시)

## 개선
- **B · 그리드 골격 항상 표시**: 날짜 칸은 즉시 그리고, 일기 표시(썸네일·마크)만 로드되면 채운다.
  로딩 중에도 카드가 비지 않고, 미세한 작은 스피너만 카드 우상단에 잠깐 표시.
- **A · 인접 월 프리페치**: 현재 달을 보는 동안 이전/다음 달을 백그라운드로 미리 받아둔다(캐시).
  다음 달로 넘기면 이미 캐시에 있어 네트워크 대기 없이 즉시 표시.

기존 인메모리 캐시(본 적 있는 달은 즉시)에 더해, **처음 넘기는 달도 골격 즉시 + 미리 준비**되도록 개선.
