# 56 · 슈퍼 관리자 + 전체 탭 '개발자도구'

## 배경
받은 의견을 확인하고, 서비스 현황과 앞으로 나올 질문을 한곳에서 보기 위한 관리자 전용 도구가 필요했다.
'근육남친'(카카오, id 28) 계정을 **슈퍼 관리자**로 지정하고, 전체 탭에 관리자에게만 보이는
**개발자도구** 진입점을 추가했다.

## 동작
- 관리자 계정으로 로그인하면 전체 탭 맨 아래에 **관리자 › 개발자도구** 항목이 보인다(일반 계정엔 숨김).
- 개발자도구 홈:
  - **가입·커플 통계** — 총 사용자, 연결된 커플/사용자, 작성된 일기, 질문 뱅크, 받은 의견 수.
  - **목록** — 받은 의견 / 질문 뱅크 미리보기로 이동.
- **받은 의견** — 사용자가 '의견 보내기'로 남긴 내용을 최신순으로. 누가·어느 화면에서·언제 보냈는지 표시.
- **질문 뱅크** — 앞으로 매일 풀에서 뽑혀 나올 질문 전체를 카테고리별로 미리보기(깊이·사용횟수·기념일 트리거 등 표시).

## 권한 처리
- User에 `role`(USER/ADMIN) 추가. 개발자도구 API(`/api/dev/**`)는 JWT로 인증하고 관리자만 접근 가능
  (일반 계정은 403). `/api/me`에 `admin` 값을 실어 전체 탭 노출을 제어한다.
- 검증: id28 관리자 200·통계 정상, 일반 계정 403 확인.
