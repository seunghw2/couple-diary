# 51. 연인 사주 보기 추가

## 요청
- 사주 허브에 "연인 사주 보기"도 추가 (내 사주처럼 연인의 개인 사주를 보는 메뉴).

## 반영
### 프론트
- 허브에 **"연인 사주 보기"** 카드 신설(내 사주 ↔ 우리 궁합 사이). 서브텍스트는 연결·생일 상태에 따라 안내("일간·오행으로 보는 {연인이름}" / "상대 생일이 등록되면 볼 수 있어요" 등).
- 개인 사주 렌더링을 **공용 컴포넌트 `PersonalSaju`**로 추출 → 내 사주/연인 사주가 같은 화면(반전 한 줄·오행으로 본 나·이런 사람이에요·강점&보완·원국) 공유. 콘텐츠 개선 시 한 곳만 고치면 됨.
- 연인 사주 화면(`/saju/partner`)은 **읽기 전용**: 생시 편집 없음, 오늘의 기운(내 하루용) 숨김, 최초 진입 로딩 연출은 동일.

### 백엔드
- `GET /api/saju/partner` — 커플 상대의 개인 사주 반환. 상대 미연결/생일 미등록이면 hasBirthday=false로 게이팅.
- 빈 결과 생성을 `emptyPersonal()`로 공용화.

## 검증
- 백엔드 clean compile / 프론트 tsc 0.
- 운영 백엔드에 유효 JWT(커플)로 `/api/saju/partner` 호출 → 연인 일간(넝쿨꽃형)·반전 한 줄·오행 해설 정상.
