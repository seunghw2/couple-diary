# 42. "봉인" 문구 순화

"봉인"이라는 표현이 딱딱하고 왜 있는지 헷갈린다는 피드백 → 개념(둘 다 답해야 열림)은 유지하고 사용자 문구만 부드럽게.

## 바뀐 문구 (동작 변화 없음)
- 답장 제출 버튼: "답장 봉인하기" → **"답장 보내기"** (아이콘 자물쇠 → 종이비행기)
- 제출 표시 태그: "봉인됨" → **"보냄"** (자물쇠 → 체크)
- 대기 안내: "답장을 봉인하면 …" → **"답장을 보내면 …"**
- 대기 상태 내 답장: "답장을 봉인해 두었어요." → **"답장을 보내 두었어요."**
- 작성 화면 안내: "답장을 봉인하면 상대도 답할 때까지 …" → **"답장을 보내면 상대가 답할 때까지 …"**
- 실패 알림: "답장을 봉인하지 못했어요" → **"답장을 보내지 못했어요"**
- 아카이브(안 열린 편지): "봉인된 채 남겨진 편지예요." → **"아직 열리지 않은 편지예요."**
