# 53. 오늘의 운세 추가 (점신 참고, 간단·재미·긍정)

## 요청
- 점신 앱처럼 '오늘의 운세' 추가하되 **더 간단·재미 위주·부정 표현 자제**.
- 서브에이전트로 기획 후 구현. 사주 활용 정도는 판단해서. 방향은 AskUserQuestion으로 확정.

## 기획(서브에이전트 3)
- 구조: 점신의 시간대탭·육각레이더·유명인·광고는 컷. 점수+한줄 → 항목운 → 행운 → 커플 요소로 간단하게.
- 사주 활용: 오늘 일진(`compute(오늘,null)`)을 무료로 계산 가능. **내 일간 오행 ↔ 오늘 일진 오행의 상생·상극**으로 점수 산출(결정론).
- 콘텐츠: 전부 긍정, 낮아도 충전·성장 프레임. '건강운'→'활력운'.

## 확정(AskUserQuestion)
- 사주 활용: **오늘 일진 × 내 사주**
- 항목: **4개 — 애정·재물·활력·행운**
- 재미: **오늘 함께하면 좋은 것 팁**(커플 요소)

## 반영
### 백엔드 (라이브 배포)
- `SajuDailyFortune`: 내 일간 오행과 오늘 일진(천간·지지) 오행의 `elemRel`로 우호도→총운, 항목 4개(상생상극 매핑 + 결정론 지터)로 60~99 밴드 점수. 상극도 60 밑으로 안 떨어지게 순화.
- 문구는 점수 대역(하/중/상)별 긍정 톤 풀. 행운색(오늘 기운 오행 연결)·아이템·키워드·숫자, 커플 '오늘 함께하기' 팁.
- 생일 없으면 일간 폴백(부스트 절반)로 편향 완화 + "생일 넣으면 맞춤 운세" 유도.
- `GET /api/saju/daily/detail`.

### 프론트
- 새 화면 `/saju/today`: 날짜+큰 점수+한줄 → 항목 4개 막대 → 행운(색·아이템·키워드·숫자) → 커플 오늘팁 → 면책.
- 허브의 '오늘의 기운' 배너를 **눌러서 상세로**(기존 배너=요약, 상세=전체. 중복 최소).

## 검증
- 백엔드 clean compile / 프론트 tsc 0.
- 운영 스모크: 항목 점수가 사람·날짜별로 다양하게 흩어짐(예 애정66·재물62·활력68·행운89), 같은 일간·같은 날은 동일(결정론).
