# 02 · 궁합 이론과 점수화 모델

> 대상: love today의 월드컵식 가벼운 '우리 궁합' 미니게임 / 입력: 두 사람 생년월일(+선택 생시) / 출력: 0~100% + 카테고리 등급 + 고정 템플릿 / 톤: "재미로 보는" (궁합 낮아도 상처 X)

---

## 1. 사주 궁합 대표 개념과 판정법

### 1-1. 오행 상생/상극
- 모든 간지는 목·화·토·금·수 중 하나. 두 사람의 오행 분포와 특히 **일간 오행 관계**를 본다.
- **상생**(목→화→토→금→수→목): 서로 북돋움 = 좋음. **상극**(목→토→수→화→금→목): 부딪힘 = 기본 나쁨.
- **뉘앙스**: 상극이 무조건 나쁜 게 아님. 한쪽 기운이 **과다**하면 눌러주는 상극이 오히려 균형(좋아짐). 한쪽에 **없는** 기운을 상대가 채우면 **상보적**(좋음).
- **판정**: 일간 상생/같은오행=좋음, 상극=기본 나쁨(단 과다 제어면 중립~좋음), 빈 오행 보완=좋음.

### 1-2. 일간(日干) 합/충 — 천간 관계
- **천간합** 5쌍(갑기·을경·병신·정임·무계): 자연스러운 강한 끌림 = 매우 좋음(부부지합).
- **천간충**(갑↔경, 을↔신, 병↔임, 정↔계): 긴장·충돌 = 나쁨(자극·활력으로 재해석도).

### 1-3. 지지 관계 — 삼합·육합·충·형·파·해·원진
- **육합** 6쌍(자축·인해·묘술·진유·사신·오미): 안정적 끌림 = 좋음.
- **삼합** 4국(해묘미·인오술·사유축·신자진): 죽이 잘 맞음 = 매우 좋음.
- **충** 6쌍(자오·축미·인신·묘유·진술·사해): 갈등 = 나쁨.
- **형/파/해**: 다툼·깨짐·방해 = 나쁨(형은 무거운 의미).
- **원진살** 6쌍(자미·축오·인유·묘신·진해·사술): '이유 없이 미운' 애증 = 나쁨(끈끈함으로 재해석도).
- **판정**: 삼합·육합=가산, 충·형=큰 감점, 파·해·원진=작은 감점.

### 1-4. 띠 궁합(년지)
- 위 지지 관계를 **년지**에만 적용한 대중적 버전. 가장 친숙, 가볍게 쓰기 좋음.

### 1-5. 음양 균형
- 두 사람 음양이 심하게 쏠리지 않고 섞이면 좋음(음+양=상보). 가벼운 보조 지표.

## 2. 앱에 넣을 것 vs 뺄 것 (커플 앱 톤)

**원칙: 밝고 상보적인 개념은 살리고, 무섭거나 '이별 확정' 식은 순화/제거.**

| 개념 | 앱 사용 | 처리 |
|---|---|---|
| 오행 상생/상극 | **사용(핵심)** | "서로 채워준다" 스토리 |
| 오행 상보(빈 기운 보완) | **사용(강추)** | 가장 따뜻한 서사 |
| 일간 천간합 | **사용(핵심)** | "운명적 끌림" 하이라이트 |
| 삼합·육합 | **사용(핵심)** | 밝고 긍정, 띠 궁합 친숙 |
| 음양 균형 | **사용(보조)** | "정반대라 더 끌려" |
| 천간충·지지충 | **순화 사용** | 감점하되 "티격태격 케미", "이별" 금지 |
| 형(刑) | **빼거나 극순화** | 무서운 뉘앙스, 이름 숨기고 미세 감점만 |
| 파·해 | **빼거나 미세 감점** | 문구 노출 X |
| **원진살** | **개념만, 이름 순화** | '살(煞)' 노출 금지, "가끔 이유 없이 삐지는 츤데레 포인트"로 |

**저점(낮은 궁합) 표현 원칙:**
1. "궁합 나쁨/이별" 단어 절대 금지.
2. 낮음 = "노력형/성장형 커플", "서로 다른 세계", "밀당의 고수들".
3. 낮은 카테고리 옆에 **높은 카테고리 하나**를 나란히 배치(위로).
4. 항상 긍정 CTA로 마무리("오늘 한마디 건네보기").
5. 최저 점수도 허용하되 문구는 100% 응원 톤.

## 3. 결정론적 궁합 점수 모델

### 3-1. 원칙
- **결정론적**: 같은 두 생일 → 항상 같은 결과(랜덤 X).
- **생시 없어도 동작**: 없으면 일간·년지·월지로, 있으면 시지까지. 생시 유무로 결과가 뒤집히지 않게.
- **점수 튜닝**: 순수 계산은 중앙 몰림 → 최종을 **60~99% 대역으로 리매핑**(커플 앱은 높게 나와야 기분 좋음). 내부 원점수 0~100은 등급 판정용으로 유지.

### 3-2. 원재료(feature)
각 사람: 일간 오행/음양, 년지(띠)·월지·일지·(선택)시지, 오행분포[목화토금수], 음양비율.
두 사람 관계: 천간합/충, 지지 육합·삼합·충·형·파·해·원진, 오행 상생/상극/상보, 합산 오행 균형도, 음양 균형.

### 3-3. 5개 카테고리 & 계산 요소 (가중치는 예시)
| 카테고리 | 컨셉 | 요소 |
|---|---|---|
| **① 첫끌림 케미** | ⚡ | 천간합(+40)·삼합(+25)·육합(+15)·일간 상생(+15)·음양 상보(+10)·천간충(−15) |
| **② 대화·소통** | 💬 | 일간 상생방향(+30)·월지 육합/삼합(+25)·오행분포 유사(+20)·지지 충(−20)·파/해(−10) |
| **③ 애정 표현** | ❤️ | 일지 육합(+35)·삼합(+25)·화(火) 총량(+15)·음양 조합(+15)·원진(−15, "츤데레"로 순화) |
| **④ 안정감·장기** | 🏡 | 일지 육합(+30)·오행 상보(+30)·전체 균형(+20)·일지 충(−25)·형(−15) |
| **⑤ 성장·밸런스** | 🌱 | 상극의 '과다 제어'(+30 긍정 재해석)·음양 균형(+25)·상보성(+25)·극단 쏠림(−20) |

> 5개가 부담되면 ①②③④ 4개로. ⑤는 낮은 점수를 "성장형"으로 완충하는 안전판이라 추천.

### 3-4. 최종 종합 점수
```
raw_total = Σ (카테고리_원점수 × 가중치)   // 예: 케미0.25 대화0.20 애정0.20 안정0.20 성장0.15
display% = round(60 + normalized(0~1) × 39)  // 60~99% 대역, 최고 99(신비감)
```
- 레어 조합(천간합 + 삼합 동시 등)이면 배지("⭐ 운명적 커플") 부여 → 월드컵식 리워드.

### 3-5. 등급 컷 (원점수 0~100)
- 상: 67~100 / 중: 34~66 / 하: 0~33. (문구는 3등급, %는 리매핑값 별도 표시)

## 4. 고정 템플릿 문구 — 개수 & 예시

### 필요 개수
- 카테고리별: 5 × 3등급 × 3개 = **45개**
- 종합(총점) 5구간 × 2 = **10개**
- 레어 배지 3~5개 + 오행 상보 특수 하이라이트 5~8개
- **합계 ≈ 65~70개**(최소만 하면 20개로도 출시 가능)

### 예시 문구(따뜻·재미 톤)
**① 첫끌림 케미** — 상: "처음 본 순간부터 '이 사람이다' 싶었죠? 자석처럼 붙는 케미예요. ⚡" / 하: "정반대 매력의 만남! 서로 신기해하며 알아가는 재미가 쏠쏠할 거예요. 🧭"

**② 대화·소통** — 상: "말하지 않아도 통하는 사이. 눈빛만 봐도 아는 커플. 💬" / 하: "대화 스타일이 좀 다르네요! 그래서 '오늘 뭐 했어?' 한마디가 더 소중한 두 분. 📞"

**③ 애정 표현** — 하: "가끔 이유 없이 삐지는 귀여운 츤데레 포인트가 있어요. 사실 그게 애정의 증거! 😤❤️"

**④ 안정감·장기** — 상: "함께 있으면 마음이 편안해지는 사이. 오래오래 갈 든든한 커플. 🏡" / 하: "서로 다른 세계에서 온 두 사람! 맞춰가는 만큼 더 특별해지는 성장형 커플. 🌱"

**⑤ 성장·밸런스** — 상: "서로의 부족한 부분을 딱 채워주는 퍼즐 같은 커플. 함께라 더 완벽해요. 🧩"

**종합** — 95+: "이건 뭐… 하늘이 정해준 짝 아닌가요? 아껴주세요 이 사람! 💍" / 60~64: "궁합은 숫자보다 노력! 만들어가는 재미가 있는 커플이에요. 화이팅! 💪"

**레어 배지** — "⭐ 운명적 커플 인증: 천간합 + 삼합이 동시에! 흔치 않아요."
